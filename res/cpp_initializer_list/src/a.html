You can use variadic functions in c++ using initializer_list.

#include<iostream>
#include<string>
#include<initializer_list>
template<class T>
void fn(std::initializer_list<T> const& list){
  for(auto elem:list){
    std::cout<<elem<<std::endl;
  }
}
int main(int argc,char** argv){
  fn({1,2});
  return 0;
}

std::initializer_list was introduced in C++11. It initializes a container type with a pre-defined element set and has a syntax similar to C arrays.

A std::initializer_list is also automatically constructed in certain cases like when a braced-init-list is used to list_initialize an object and the corresponding constructor accepts std::initializer_list as a parameter as in the above. It is also created when a abraced-init-list is used as the rvalue of an assignment and the assignment operator or function accepts std::initializer_list as parameter. Also, when braced-init-list is bound to auto an object of type std::initializer_list is constructed.

initializer_list does not have a subscript operator, but you can use something like list.begin()[0] for example.

std::initializer_list can be used int constructors as follows.

#include<initializer_list>
class Point{
  public:
    Point(std::initializer_list<int> il){
      mx=il.size()>0?il.begin()[0]:0;
      my=il.size()>1?il.begin()[1]:0;
      mz=il.size()>2?il.begin()[2]:0;
    };
  private:
    int mx;
    int my;
    int mz;
};

More succinctly, you can pass std::initializer_list to a member's constructor, for example

#include<initializer_list>
#include<vector>
class List{
  public:
    Point(std::initializer_list<std::string> il):mdata(il){};
  private:
    std::vector<std::string> mdata;
};
