Lambdas were introduced in C++11 and allows you to write inline functions. They can be used for short snippets of code that you are not going to reuse, for example in algorithms.

Lambdas in C++ provides a ligter syntax than functors. Everything you can do with a lambda you can do with a functor.

Important parts include the capture clause for controlling scope, parameters for specifying the parameters that will be used by the lambda, the return type that specifies the return type, and the code which goes inside the lambda. Here is an arbitrary lambda example

#include<iostream>
int main(int argc,char** argv){
  auto a=[](){
    std::cout<<"test"<<std::endl;
  };
  a();
  return 0;
}

You can specify the scope within the [] part.

  • []: no captures, only local variables
  • [&]: capture all externals
  • [a,&]: capture a by value and b by reference

If there are no captures, then the following will be invalid

int a=0;
auto b=[](){
	std::cout<<a<<std::endl;
};
b();

In order to use a in the lambda the following serves as illustration

int a=0;

[&](){
	std::cout<<a<<std::endl;
}();

[a](){
	std::cout<<a<<std::endl;
}();

[&a](){
	std::cout<<a<<std::endl;
}();

The return type is normally inferred but can be explicitly set as in the following example

std::cout<<([]()->int{return 0;}())<<std::endl;

Lamdas are often used in algorithms, for example

std::vector a={0,1,2,3};
std::for_each(a.begin(),a.end(),[](int b){
	std::cout<<b<<std::endl;
});
